## Overview

**Shantighar Website** is a public-facing website designed for the Shantighar Social Service Society, supporting women and children in crisis. I developed this platform during my internship as a Web Development Intern, where I focused on responsiveness, cross-browser compatibility, and accessibility.

### Key Contributions

- **Responsive Design**: Created a mobile-friendly UI using HTML, CSS, and JavaScript to ensure usability on all device sizes.
- **Cross-Browser Compatibility**: Followed industry best practices to make the site consistent across various browsers.
- **Performance Optimization**: Enhanced website performance by implementing clean structure and optimized code.
- **Social Impact**: Helped improve visibility and access to social services provided by Shantighar.
